## ✨ Features

- **BIOS Boot Sequence** — Animated startup screen with memory test and loading bar
- **Draggable Windows** — Every section opens as a movable, stackable window
- **Live Clock** — Real-time clock and date in the taskbar
- **Taskbar & Desktop Icons** — Full OS-like navigation
- **Fully Responsive Data** — Projects link directly to GitHub repos
- **Zero Dependencies** — Pure HTML, CSS & JavaScript, no frameworks

## 🚀 Run Locally

No installation needed — just open the file in your browser:

```bash

cd Gravity-2010.github.io
open index.html   # macOS
# or
start index.html  # Windows
```
